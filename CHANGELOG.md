# 更新日志

## [1.1.0] - 2026-02-17

### 🎉 新增功能
- **隐私模式** - 在搜索框输入"开门"可显示隐藏的分类和链接
  - 支持设置分类和链接为私密
  - 显示隐私模式提示横幅
  - 提供一键退出功能
  - 保持搜索功能正常工作

### 🔧 技术改进
- 添加 `isPrivate` 字段到类型定义（`NavLink`, `NavCategory`）
- 优化搜索框组件，添加"开门"输入提示
- 添加淡入动画效果
- 改进数据过滤逻辑

### 📝 文档更新
- 简化文档结构，整合为 README.md 和 CHANGELOG.md
- 添加隐私模式使用说明
- 更新项目特性和功能列表

## 🚀 接下来怎么做？

### 方案 1：纯网页操作（推荐，不需要命令行）

1. **阅读部署指南**
   - 打开 `Cloudflare网页部署指南.md`
   - 或者看 `快速部署.md`（更简洁）

2. **在 Cloudflare 网页端操作**
   - 创建 D1 数据库（复制 `schema.sql` 内容）
   - 创建 Worker（复制 `cloudflare/src/index.js` 内容）
   - 修改 `nav.html` 中的 API 地址
   - 部署前端到 Pages（上传 `nav.html` 等文件）

3. **完成！**

### 方案 2：使用 GitHub + 命令行

1. **初始化 Git**（双击运行）
   - Windows: `init-git.bat`
   - Mac/Linux: `bash init-git.sh`

2. **上传到 GitHub**
   ```bash
   # 在 GitHub 创建仓库后运行：
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

3. **部署到 Cloudflare**
   - 按照 `快速部署.md` 操作

## ⚠️ 重要提醒

### 1. 修改 API 地址

部署 Worker 后，**必须**修改 `nav.html` 中的 API 地址：

```javascript
// nav.html 第 446 行左右
const API_BASE_URL = 'https://你的-worker-url.workers.dev/api';
```

### 2. 数据库初始化

在 D1 控制台执行 `cloudflare/schema.sql` 的内容（包含表结构和示例数据）

### 3. Worker 绑定数据库

Worker 的 Settings → Bindings → 添加 D1 database：
- Variable name: `DB`（必须大写）
- D1 database: 选择你创建的数据库

### 4. 默认密码

- 管理密码：`admin123`
- **部署后立即修改！**
